# 项目简报：梦境造物主 (Dream Weaver)

## 归一化字段

- **projectName**: Dream Weaver / 梦境造物主
- **tagline**: 孩子是故事的主角
- **description**: AI 儿童绘本创作平台：上传孩子照片生成专属卡通形象，输入关键词与教育目的即可生成个性化绘本与有声书，支持父母声音克隆朗读。
- **primaryFeatures**:
  - 角色定制：上传孩子照片，AI 提取特征生成卡通形象作为故事主角
  - 剧情共创：输入关键词（如恐龙、太空、不想刷牙）与教育目的（如学会分享、克服恐惧）
  - 实时绘本生成：约 5 分钟短故事 + 每段配图，风格统一（水彩风、皮克斯风），主角像孩子
  - 有声书模式：克隆父母声音自动朗读，父母不在家孩子也能听爸爸妈妈讲故事
- **targetUsers**: 家长、学龄前及低年级儿童
- **domain**: aiaudiotools.shop
- **appUrl**: https://aiaudiotools.shop
- **defaultLocale**: zh
- **supportedLocales**: [zh, en]
- **socialLinks**: supportEmail = support@aiaudiotools.shop（TODO: 可替换为实际邮箱）
- **branding**:
  - **primaryColor**: #8B7EC8（柔和紫，梦境/儿童友好）
  - **logoAssetPath**: 无，使用脚本生成
  - **preferSvg**: false（v1 保持 logo.png / favicon.ico）
